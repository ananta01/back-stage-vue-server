# 电费在线管理系统（服务端）

## Project setup
```
npm install
and
github back-stage-vue(客户端)
```

### Compiles and hot-reloads for development
```
npm run server
```

## 项目介绍
##### 电费在线管理系统采用前后端分离的模式进行开发，前端技术栈（Vue全家桶 +　ElementUI）,后端技术栈（Node.js(express框架) + Mongodb数据库）。

## 数据模块
* 用户模块（User.js）
* 用电查询（MonthFile.js）
* 用电明细（ProFile.js）
* 用户留言（UserMes.js）
* 系统公告（SysNotice.js）
